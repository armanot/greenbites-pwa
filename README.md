# GreenBites PWA Starter

Starter MVP untuk GreenBites:
- Dashboard ringkas
- Snap Result screen
- PWA installable
- Simpan log harian dalam `localStorage`

## Cara jalan

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy ke Vercel
1. Push repo ke GitHub
2. Import repo dalam Vercel
3. Deploy
